Sinamon = Sinatra + Mongoid
=====
A boilerplate for simple web project

![Logo](http://usefulpa.s3.amazonaws.com/images/2014/cinnamon_roll.png)

Usage
-----

	$ git clone && cd
	$ bundle
	$ foreman start
