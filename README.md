Sinamon = Sinatra + Mongoid
=====
A frontend-agnostic server-only webapp framework based on Sinatra and Mongoid

![Logo](http://usefulpa.s3.amazonaws.com/images/2014/cinnamon_roll.png)

Usage
-----

	$ git clone && cd
	$ bundle
	$ thin start
    $ open http://localhost:5000
